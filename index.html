FM2022 Script

A FM2022 save game is dictated heavily by behind the scenes attributes, this tool automates the lengthy process of changing these attributes to create a dynamically changing game. Data is read in through a .txt file. The script loops through the data and uses mouse & keyboard inputs to perform actions
